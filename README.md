# Codecademy-Open-Source
> Creating a test repository for Codecademy’s Open Source course!

Ce dépôt a été créé dans le cadre du cours **Open Source** de Codecademy.  
Il sert principalement de bac à sable pour expérimenter les différentes étapes du workflow open‑source : création de branches, ouverture de pull requests, gestion des issues, collaboration, et bonnes pratiques Git/GitHub.

## Objectifs du projet

- Comprendre les bases de la contribution open source  
- Se familiariser avec GitHub et son écosystème  
- Tester différentes commandes Git en situation réelle  
- Apprendre à documenter correctement un dépôt

## Contenu du dépôt

Ce repository peut contenir :
- Des fichiers d’exercices liés au cours
- Des essais de fonctionnalités Git
- Des exemples de documentation ou de contributions

## Comment contribuer

Ce dépôt est conçu pour la pratique : n’hésite pas à créer une branche, proposer une modification ou ouvrir une issue dans le but d’appliquer les concepts du cours.
